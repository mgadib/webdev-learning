# Webdev Learning

Platform pembelajaran web development interaktif yang menyediakan modul pembelajaran terstruktur dengan materi chapter, progress tracking, dan kanvas interaktif untuk pengalaman belajar yang menyenangkan.

## Fitur

- **Modul Pembelajaran**: Akses modul-modul pembelajaran web dev yang terstruktur
- **Chapter & Materi**: Pelajari materi per chapter dengan konten pedagogis lengkap
- **Progress Tracking**: Lacak progress belajar dengan local storage
- **Refleksi**: Simpan catatan refleksi per chapter secara otomatis
- **Animasi Interaktif**: Kanvas interaktif dan animasi custom untuk pengalaman belajar yang unik

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- React Router

## Pengembangan

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
